### MEM React App for studies in WSB

> React Stack

- material-ui
- formik / yup
- react-router-dom
- react-loader-spinner
- redux

[Hosted on gh-pages](https://inga.maczka/)

> Usage

1. `git clone https://maczka.inga` - Clone Repo
2. `yarn install` - Install Dependencies
3. `npm start` - Enjoy :smiling_imp:
