### MEM React App for studies in WSB

> React Stack

- react-router-dom
- redux
- redux-thunk
- material-ui
- formik / yup
- styled-components

---

> Usage

1. `git clone https://github.com/maczking/react_mem_WSB` - Clone Repo
2. `yarn install` - Install Dependencies
3. `npm start` - Puff, now you can see it
